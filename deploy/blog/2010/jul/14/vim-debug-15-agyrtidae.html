Vim Debug 1.5: Agyrtidae

Posted: Wed, 14 Jul 2010

Agyrtidae is the name of a family of Beetles (found in Russia), and the codename for the 1.5 release of Vim Debug.

Features of this release

  • python support!!
  • really simple startup
    • no more quickly switch over to start your debug server within the 5 seconds
    • auto-start current file
    • auto-start a url (for PHP)
    • remember the last 5 urls started
  • huge code redesign
  • better help for commands
  • live stack view
  • watch expressions

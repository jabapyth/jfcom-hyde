Integrated python debugging in VIM

Posted: Mon, 12 Jul 2010

A few days ago, I posted that I was taking over a project which provides a debugging environment in VIM, specifically geared toward PHP. Soon after a guy w/ the handle Epeli asked (perhaps not very grammatically), “I can we debug Python like this with Vim?”

but we’ll forgive the grammar because he’s from Finland, and I can’t pronounce the name of his home town…

But it was an awesome question! I thought, well why not? The debugger is really just interfacing with a server which conforms to the DBGp specification, and after looking around I was happy to find that ActiveState had done most of the legwork for me. Unfortunately, there were a few show-stopping bugs in their code, and I couldn’t find it anywhere on version control (though it is open source), so I took the liberty of throwing it up on github and added my patches — so now it’s ready to use!

[btw if it is on a VCS somewhere and I missed it, let me know so I can send my fixes upstream]

That’s right; we now have integrated debugging in your favorite power tool (well, mine anyway): VIM.

Here’s an annotated screenshot of an active session, showcasing:

  • breakpoints
  • watch expressions
  • execution stack view
  • current scope
  • output stream copying

On the github project page, you can find install instructions for the python server. I’ll soon put together install instructions for vim-debug.

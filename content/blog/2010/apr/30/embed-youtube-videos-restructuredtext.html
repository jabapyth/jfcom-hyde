--- title: 'Embed youtube videos in RestructuredText' slug: embed-youtube-videos-restructuredtext created: !!timestamp '2010-04-30 06:30:26' extends: blog-rst.j2 status: public --- {% mark exerpt %} I write this blog almost exclusively in `RestructuredText `_ and I recently wanted to embed a youtube video on the `Baby Tux project page `_. Fortunately someone's already done the work of making a directive, so you can just save the code `here `_ to ``youtube_rst.py`` and put {% endmark %} .. code-block:: python import youtube_rst in the ``__init__.py`` of your django project. Using the directive is dead simple: .. code-block:: rst .. youtube:: 9ZWyv9icIfI Cheers.

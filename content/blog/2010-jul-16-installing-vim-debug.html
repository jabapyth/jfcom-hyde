--- title: 'Installing Vim Debug' slug: installing-vim-debug created: !!timestamp '2010-07-16 13:13:07' extends: blog-rst.j2 status: public tags: - vim-debug - vim - debug - installation - how-to --- {% mark excerpt %} I've had a few people ask me for installations for `vim-debug `_, which I guess is natural...I'm just not used to the whole "package maintainer" persona yet. Anyway, here it is. {% endmark %} {% mark image %} .. image:: http://jaredforsyth.com/media/uploads/images/install.png {% endmark %} .. code-block:: bash sudo pip install dbgp sudo pip install vim-debug install-vim-debug.py .. note:: To start your debugging session, you need a *capital* D. ``:Dbg``. While you are in the session, you can use a lowercase, ex: ``:dbg break`` Those three commands should do it for you. I tested it on a completely fresh fedora 13 install, and devved on Ubuntu Karmic I have no idea whether this will work on windows -- if you try it, be sure to tell me! Thanks
